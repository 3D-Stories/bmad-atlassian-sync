# BMAD Atlassian Sync

Bidirectional sync between BMAD .md artifacts and Jira Cloud / Confluence Cloud via REST APIs.

**Usage:** This skill is included by other BMAD skills (sprint-planning, create-story, dev-story, etc.) to sync their outputs with Jira and Confluence. It can also be invoked directly for manual sync operations.

**Standalone CLI:** For non-BMAD projects, use `atlassian-sync` CLI directly.

Read fully and follow: `./workflow.md`
